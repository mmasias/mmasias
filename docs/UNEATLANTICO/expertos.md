# Acerca de los expertos

> *Hay una diferencia entre mapa y territorio...*

- **Definición 1:** Es un experto S sobre un dominio D quien tiene un mayor número de creencias verdaderas y menor número de creencias falsas respecto a D que la mayoría de la gente. Además, el número de creencias verdaderas debe ser sustancial.

  - *Centra el expertise en la posesión cuantitativa de creencias verdaderas sobre D. Requiere dos condiciones: superioridad relativa respecto a la mayoría (más verdades, menos falsedades) y masa crítica absoluta (número sustancial de creencias verdaderas).*
  - Esta definición se corresponde al modelo de experto como ***poseedor de conocimiento disciplinar*** (investigador, especialista).

- **Definición 2:** Es un experto S sobre un dominio D quien tiene la capacidad de ayudar a otros (especialmente personas no expertas) a resolver una variedad de problemas en D o ejecutar una variedad de tareas en D, que las personas no iniciadas no podrían resolver o ejecutar por su cuenta.

  - *Centra el expertise en la capacidad de transferencia operativa. El experto se define por poder habilitar a no-expertos para resolver problemas o ejecutar tareas que, sin esta intervención, permanecerían inaccesibles.*
  - Esta definición se corresponde al modelo de experto como ***facilitador de competencias*** (docente efectivo, consultor operativo).

## Arquetipos

<div align=center>

|Arquetipo|Definición|Ejemplo|
|-|-|-|
**Experto completo**|Domina conocimiento disciplinar sustancial y puede transferirlo operativamente, habilitando a otros para resolver problemas variados que no podrían abordar sin intervención.|Profesor-investigador que forma estudiantes capaces de resolver problemas nuevos en su disciplina.
**Facilitador pragmático**|Posee frameworks metodológicos o conocimiento procedimental que permiten ayudar a otros a resolver problemas, sin dominar sustancialmente el conocimiento disciplinar subyacente.|Coach ágil que facilita resolución de problemas de gestión sin expertise técnico en el dominio.
**Teórico inerte**|Posee conocimiento disciplinar sustancial pero carece de capacidad para transferirlo operativamente. No puede habilitar a otros para resolver problemas, aunque él mismo comprenda el dominio.|Investigador con publicaciones que no logra formar estudiantes competentes.
**No-experto**|Carece tanto de conocimiento disciplinar sustancial como de capacidad para facilitar resolución de problemas en el dominio.|Persona ajena al dominio.

</div>
