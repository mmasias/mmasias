# Variables

---
Espacio en memoria reservado para almacenar un valor, que puede cambiar durante la ejecución del programa.

---
### Declaración
<TIPO><VARIABLE>[=<expresión>];
```
int miOrden;
double miCosto;
char unCaracter;
boolean sigueVivo;
```

---
### Asignación
<VARIABLE>=<expresión>;
```
miOrden = 30;
miCosto = 225.56;
unCaracter = 'x';
sigueVivo = true;
```

---
### Problema

Calcular el promedio de dos calificaciones
